Setup di Perangkat Mobile

Akses email kapan saja, di mana saja. Mari hubungkan akun ke aplikasi Mail di iPhone & Android.

Panduan Setup di iPhone / iPad

Langkah Pengaturan Awal

  1. Buka Settings (Pengaturan).
  2. Scroll dan pilih Mail.
  3. Pilih Accounts (Akun).
  4. Pilih Add Account (Tambah Akun).
  5. Pilih Other (Lainnya) dari daftar.
  6. Pilih Add Mail Account (Tambah Akun Mail).

Detail Akun & Server

1. Isi Nama, Email (`hello@gaianata.id`), dan Password baru. Klik Next.

2. Pilih tipe akun IMAP.

3. Masukkan detail server IMAP & SMTP dari slide konfigurasi.

  • Host Name (Incoming): `imap.hostinger.com`
  • Host Name (Outgoing): `smtp.hostinger.com`
  • User Name (keduanya): `hello@gaianata.id`
  • Password (keduanya): Password baru

4. Klik Next, lalu Save untuk menyelesaikan.

Panduan Setup di Android (via Gmail)

Langkah Pengaturan Awal

  1. Buka aplikasi Gmail.
  2. Ketuk ikon profil di kanan atas, pilih Add another account.
  3. Pilih Other (Lainnya).
  4. Masukkan alamat email lengkap.
  5. Ketuk Manual Setup.
  6. Pilih IMAP sebagai tipe akun.

Detail Server

1. Masukkan Password baru, klik Next.

2. Pada Incoming server settings, pastikan detailnya benar, klik Next.

Server: `imap.hostinger.com`

3. Pada Outgoing server settings, pastikan detailnya benar, klik Next.

SMTP Server: `smtp.hostinger.com`

4. Atur opsi sinkronisasi, lalu beri nama pada akun untuk menyelesaikan.

Setup Lanjutan: Integrasi Gmail

Kelola semua email (pribadi dan profesional) dalam satu tempat, langsung dari antarmuka Gmail di komputer.

Penting: Proses ini harus dilakukan melalui Browser Desktop/Laptop.

Cara Mengirim & Menerima Email dari Gmail

Langkah 1: Menambahkan Alamat Email

  1. Login ke akun Gmail pribadi di browser.
  2. Buka Settings ⚙️ → See all settings.
  3. Pilih tab Accounts and Import.
  4. Di bagian "Send mail as", klik "Add another email address".
  5. Sebuah jendela pop-up akan muncul.

Langkah 2: Konfigurasi Server (SMTP)

  1. Di pop-up, masukkan nama dan email `hello@gaianata.id`, lalu klik Next.
  2. Isi detail SMTP dari slide konfigurasi (Server, Username, Password, Port 465, SSL).
  3. Klik Add Account.
  4. Gmail akan mengirim email verifikasi ke `hello@gaianata.id`.
  5. Buka Webmail Hostinger, buka email tersebut, dan klik link konfirmasi.
